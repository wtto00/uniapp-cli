# uniapp-cli

CLI for uni-app

## 🚧 正在开发中...

## Features

- 🔀 CI/CD 友好，流水线中轻松打包 APP
- 🚤 完全舍弃 HBuilderX，一致的开发体验
- 🏗️ 快速新建项目
- 🔎 检查开发环境完整性
- ⏩ 快速启动项目，并支持热更新
- 📦 一键打包各平台
- 🔄 转换 HBuilderX 项目到 CLI 项目

## 安装

```shell
npm i -g uniapp_cli
```

## 用法

```shell
uniapp --help
```

更多用法参见 [uniapp-cli 文档](https://wtto00.github.io/uniapp-cli/)
